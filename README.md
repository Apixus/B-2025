# B站自动字幕

一个便捷的B站字幕控制用户脚本，支持快捷键开启和自动开启字幕。

## 功能特点

- 🎯 **快捷键控制**: 按 `C` 键快速开启字幕
- 🔄 **自动开启**: 切换分P时自动打开字幕
- 🆕 **最新适配**: 专为B站2025最新版优化
- ⚡ **轻量高效**: 不占用额外资源，随用随开

### 第二步：安装脚本

点击这里安装: [安装脚本](https://github.com/apixus/)

## 使用方法

1. 访问任意B站视频页面
2. 按 `C` 键即可切换字幕显示
3. 切换分P时会自动开启字幕
